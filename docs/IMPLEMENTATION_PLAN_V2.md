# 🚀 UCSI 챗봇 기능 완성 로드맵 (V2.4)

## Phase 1: 핵심 백엔드 기능 강화 (진행 중)
- [ ] **FAQ 캐싱 시스템**: 자주 묻는 질문(규칙 기반) 즉시 응답 처리 (AI 비용 절감 및 속도 향상)
- [ ] **미응답 질문 관리**: 답변하지 못한 질문 로깅 및 관리자 검토 큐 생성
- [ ] **피드백 기반 학습**: 사용자 피드백(좋아요/싫어요)을 반영한 응답 개선 로지고
- [ ] **다국어 지원 엔진**: 한국어/영어/중국어 자동 감지 및 언어별 맞춤 응답

## Phase 2: 데이터 조회 기능 확장
- [ ] **학생 등록 상태 조회**: 현재 학기 등록 여부 확인 API 연동
- [ ] **수강 프로그램/학기 상태**: 수강 중인 과목 및 학기 정보 상세 조회
- [ ] **교수진/학과 상세 소개**: DB 데이터를 활용한 상세 정보 제공 카드 UI

## Phase 3: 관리자 기능 구현 (Web UI)
- [ ] **관리자 대시보드 UI**: HTML 기반 관리자 전용 페이지
- [ ] **문서 업로드 기능**: PDF/Word 드래그 앤 드롭 업로드 및 자동 RAG 벡터화
- [ ] **정보 수정 기능**: 정책, 프로그램 정보 등을 웹에서 직접 수정
- [ ] **통계 시각화**: 피드백, 자주 묻는 질문, 미응답 현황 차트

## Phase 4: 사용자 편의 기능
- [ ] **대화 히스토리 내보내기**: 상담 내용을 PDF로 다운로드하는 기능

## Phase 5: 마무리 및 최적화
- [ ] **코드 리팩토링**: 모듈화 및 클린 코드 적용
- [ ] **문서 현행화**: 한글 문서 최신화 (README, SPECS 등)
- [ ] **파일 정리**: 불필요한 테스트 파일 및 임시 파일 삭제
